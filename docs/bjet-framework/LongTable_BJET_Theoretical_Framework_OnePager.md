# LongTable의 BJET 제출용 이론 프레임워크 1페이지 초안

## 1. 핵심 주장

LongTable는 "사람처럼 말하는 AI"가 아니라, 학습자가 서로 다른 관점과 근거를 탐색하고 비교하며 스스로 판단하게 만드는 **human-mediated epistemic tool**로 설계되어야 한다. BJET용 핵심 주장은 다음과 같다.

> 학습에서 생성형 AI의 교육적 가치는 자동화 그 자체가 아니라, 인간이 개입하여 관점의 다양성, 대화적 성찰, 판단의 책임을 구조화할 때 강화된다.

## 2. 코어 이론 앵커

이 프레임워크는 많은 이론을 병렬로 얹지 않고, 아래 세 층으로 고정한다.

| 층위 | 역할 | BJET에서의 기능 |
|---|---|---|
| **핵심 설명 이론** | **Dialogic pedagogy** | 학습을 정답 전달이 아니라 관점 간 대화, 긴장, 응답성의 과정으로 본다. |
| **핵심 학습 메커니즘** | **Epistemic agency** | 학습자가 AI의 출력을 수동 소비하지 않고 선택, 비교, 반박, 재구성한다. |
| **핵심 설계 가치** | **Human mediation / value-sensitive design** | 어떤 관점을 제시할지, 어떤 근거를 드러낼지, 무엇을 보류할지에 인간의 책임 있는 개입을 남긴다. |

## 3. 개념 구조도

```text
Human mediation
  -> 관점 선택과 편집의 가시화
  -> 근거와 출처의 명시
  -> 반론 가능성(contestability)의 제공

  => Dialogic divergence
     서로 다른 관점, 프레임, 가치, 근거를 병치

  => Epistemic agency
     학습자의 비교, 반박, 통합, 재판단

  => Educational outcomes
     더 깊은 성찰, 더 나은 판단 정당화, 과도한 AI 의존 감소
```

## 4. 왜 BJET에 맞는가

- BJET는 교육기술이 실제로 **어떻게 학습을 개선하는지**를 이론적, 방법론적, 경험적으로 보여주는 논문을 선호한다.
- 따라서 LongTable의 기여는 "새로운 AI 기능"보다 **인간 매개를 통해 학습의 질을 어떻게 재설계하는가**에 있어야 한다.
- 이때 AI의 인간 유사성보다는, **대화적 학습을 촉진하는 교육 설계 원리**가 더 중요하다.

## 5. 이 프레임워크가 요구하는 연구 질문

1. 인간이 매개한 divergence 설계는 학습자의 epistemic agency를 어떻게 촉진하는가?
2. 학습자는 서로 다른 AI-제안 관점과 인간 편집 흔적을 어떻게 해석하고 활용하는가?
3. 이런 설계는 학습자의 성찰적 판단, 근거 기반 토론, AI 의존 방식에 어떤 변화를 만드는가?

## 6. BJET용 권장 방법

- **1단계: 질적 탐색**
  포커스 그룹, 전문가 패널, 교사 인터뷰로 "어떤 종류의 관점 다양성"과 "어떤 인간 개입"이 교육적으로 의미 있는지 도출한다.
- **2단계: 설계기반연구(DBR)**
  초기 프로토타입을 반복 개선하며 설계 원리와 사용 맥락을 정교화한다.
- **3단계: 경험적 검토**
  학습자 로그, 토론 산출물, 인터뷰, 자기보고를 결합해 agency와 성찰이 어떻게 나타나는지 분석한다.

## 7. 반드시 피해야 할 것

- agency, trust, engagement, anthropomorphism, diversity를 한 논문에 모두 넣는 것
- 검증되지 않은 새 척도를 여러 개 동시에 만드는 것
- AI가 "사람 같아 보이는가"를 인간 개입의 증거로 착각하는 것
- 교육적 기여보다 시스템 기능 설명이 앞서는 것

## 8. 한 문장으로 요약

BJET용 LongTable는 **AI의 답변 품질**을 증명하는 논문이 아니라, **인간이 매개한 다원적 대화 구조가 학습자의 epistemic agency를 어떻게 강화하는지**를 보여주는 논문이어야 한다.
